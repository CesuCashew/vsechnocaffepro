# Všechno Caffe 2.0

Premium website for Všechno Caffe - Music Bar, Restaurant & Café in Cheb.

## Features

- **Time-based Themes**: Automatically switches between Café (morning), Lunch (afternoon), and Club (evening) modes
- **Premium Design**: Glassmorphism effects, sophisticated typography, and smooth animations
- **Responsive**: Works on all devices
- **Interactive**: Scroll animations, parallax effects, and dynamic content

## Technologies

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Outfit)
- Font Awesome Icons

## Local Development

Simply open `index.html` in your browser. No build process required.

## Structure

```
vsechnocaffe2/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # All styles
├── js/
│   └── main.js        # JavaScript logic
└── img/               # Images and assets
```
